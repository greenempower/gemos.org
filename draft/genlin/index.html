Draft / GenLin (Gen Linux) tool

This is the draft for a tool named Gen Linux. It's purpose is to prepare a customizable Linux image for hosting GEMx.

This tool should be a much more compact and easy to use solution to Yocto. It should favor simple descriptions and interactive elements.
